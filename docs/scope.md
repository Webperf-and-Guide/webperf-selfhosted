# Scope

This repository focuses on a self-hosted workflow for release verification.

## Included

- manual checks
- saved properties, route sets, region packs, and check profiles
- scheduled dispatch via external cron or webhooks
- baseline comparison and latest-vs-previous comparison
- report generation and export
- generic webhook alerting
- Docker Compose deployment

## Not Included

- multi-tenant billing
- enterprise identity and seat management
- provider-managed global orchestration
- private admin surfaces
- provider-specific premium routing guarantees
